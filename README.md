# 소개
> 이 클론코딩 강의를 통해 React에 한 발자국 나아갈수 있었습니다.
> 자바스크립트 라이브러리의 중요성을 알게되었고, 응용을 할수 있게 되었습니다.
> 또한 앞으로 실무에서 사용될 Typescript를 공부 하였습니다.

## 실행 전 세팅
```
$ npm init
$ ~
...
```

## 실행
```
$ npm run dev
$ yarn run dev
```
